The Problem
---

AI coding agents forget everything between sessions. Every time you start a new session, the agent starts from scratch:

<ul class="staggered-list">
  <li>"Why did we choose X over Y?"</li>
  <li>"What was that bug fix again?"</li>
  <li>"What patterns does this project use?"</li>
</ul>

<div class="comparison-grid">
  <div class="before-box morph-border fade-in-left">
    <h3>Frustrating</h3>
    <p>You have to explain the same context every session</p>
  </div>
  <div class="before-box morph-border fade-in-right">
    <h3>Wasteful</h3>
    <p>Agent spends time re-learning instead of coding</p>
  </div>
</div>

---

## The Solution

toon-memory provides persistent memory that survives restarts. It stores your decisions, patterns, bugs, and knowledge in a token-efficient format.

<div class="comparison-grid">
  <div class="after-box morph-border fade-in-left">
    <h3>Efficient</h3>
    <p>Agent remembers everything automatically</p>
  </div>
  <div class="after-box morph-border fade-in-right">
    <h3>Fast</h3>
    <p>Instant recall, no re-learning needed</p>
  </div>
</div>

---

## Demo: Before & After

### Without toon-memory

<div class="before-box fade-in-up">
  <p><strong>Session 1:</strong></p>
  <div class="terminal">
    <span class="prompt">User:</span> "We use Zod for validation"<br />
    <span class="prompt">Agent:</span> "Got it!"
  </div>

  <p><strong>Session 2 (new session):</strong></p>
  <div class="terminal">
    <span class="prompt">User:</span> "What validation library do we use?"<br />
    <span class="prompt">Agent:</span> "I don't know, let me check the code..."
  </div>
</div>

### With toon-memory

<div class="after-box fade-in-up">
  <p><strong>Session 1:</strong></p>
  <div class="terminal">
    <span class="prompt">User:</span> "We use Zod for validation"<br />
    <span class="prompt">Agent:</span> <span class="command">memory_remember</span>(category="pattern", key="validation")<br />
    <span class="output">Saved: pattern/validation</span>
  </div>

  <p><strong>Session 2 (new session):</strong></p>
  <div class="terminal">
    <span class="prompt">User:</span> "What validation library do we use?"<br />
    <span class="prompt">Agent:</span> <span class="command">memory_recall</span>(query="validation")<br />
    <span class="output">[pattern] validation — We use Zod for validation</span>
  </div>
</div>
