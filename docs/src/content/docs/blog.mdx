## TL;DR

toon-memory gives your AI agent persistent memory across sessions. No more repeating context every time you start a new session.

---

## The Problem

Every time you start a new coding session, your AI agent forgets:

<ul class="staggered-list">
  <li>Design decisions you've made</li>
  <li>Patterns used in the project</li>
  <li>Bugs you've already fixed</li>
  <li>Project-specific knowledge</li>
</ul>

<div class="comparison-grid">
  <div class="before-box morph-border fade-in-left">
    <h3>Frustrating</h3>
    <p>You have to explain the same context every session</p>
  </div>
  <div class="before-box morph-border fade-in-right">
    <h3>Wasteful</h3>
    <p>Agent spends time re-learning instead of coding</p>
  </div>
</div>

---

## The Solution

toon-memory stores this context in a token-efficient format (TOON) that your agent can recall in future sessions.

<div class="comparison-grid">
  <div class="after-box morph-border fade-in-left">
    <h3>Efficient</h3>
    <p>Agent remembers everything automatically</p>
  </div>
  <div class="after-box morph-border fade-in-right">
    <h3>Fast</h3>
    <p>Instant recall, no re-learning needed</p>
  </div>
</div>

---

## Demo: Before & After

### Without toon-memory

<div class="before-box fade-in-up">
  <p><strong>Session 1:</strong></p>
  <div class="terminal">
    <span class="prompt">User:</span> "We use Zod for validation"<br />
    <span class="prompt">Agent:</span> "Got it!"
  </div>
  
  <p><strong>Session 2 (new session):</strong></p>
  <div class="terminal">
    <span class="prompt">User:</span> "What validation library do we use?"<br />
    <span class="prompt">Agent:</span> "I don't know, let me check the code..."
  </div>
</div>

### With toon-memory

<div class="after-box fade-in-up">
  <p><strong>Session 1:</strong></p>
  <div class="terminal">
    <span class="prompt">User:</span> "We use Zod for validation"<br />
    <span class="prompt">Agent:</span> <span class="command typing-effect">memory_remember(category="pattern", key="validation", content="Uses Zod")</span><br />
    <span class="output">🧠 Saved: pattern/validation</span>
  </div>
  
  <p><strong>Session 2 (new session):</strong></p>
  <div class="terminal">
    <span class="prompt">User:</span> "What validation library do we use?"<br />
    <span class="prompt">Agent:</span> <span class="command typing-effect">memory_recall(query="validation")</span><br />
    <span class="output">[pattern] validation<br />We use Zod for validation (saved in session 1)</span>
  </div>
</div>
