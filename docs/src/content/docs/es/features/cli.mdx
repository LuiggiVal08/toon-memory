title: Comandos CLI

---

## Comandos

```bash
npx toon-memory              # Instalador interactivo
npx toon-memory init         # Configuración rápida (sin preguntas)
npx toon-memory mcp          # Ejecuta el servidor MCP directamente
npx toon-memory status       # Ver estado de la instalación
npx toon-memory stats        # Ver estadísticas de memoria
npx toon-memory export       # Exportar memoria a JSON
npx toon-memory import <file> # Importar memoria desde JSON
npx toon-memory watch [opts] # Auto-backup con opciones
npx toon-memory upgrade      # Actualizar a la última versión
npx toon-memory uninstall    # Eliminar de todos los agentes
```

## Opciones de watch

| Opción | Descripción | Por defecto |
|--------|-------------|---------|
| `[interval]` | Intervalo de backup en minutos | 5 |
| `-c, --compress` | Habilita compresión gzip | off |
| `-l, --log [path]` | Habilita registro en archivo | off |
| `-m, --max-backups <n>` | Máximo de backups a mantener (0=ilimitado) | 10 |

### Export

```bash
$ npx toon-memory export

🧠 toon-memory export

  /ruta/al/proyecto/toon-memory-export.json
```

### Import

```bash
$ npx toon-memory import backup.json

Omitidas 2 duplicadas
```
